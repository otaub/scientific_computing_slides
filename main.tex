\begin{frame}{CSC's HPC Resources}
CSC's national HPC resources are free-of-charge for academic institutions, state research institutes, or the National Archives of Finland. The services discussed here are:
\begin{columns}
\column{0.3\textwidth}
    \begin{itemize}
        \item HPC clusters
        \begin{itemize}
            \item \textcolor{gray}{Puhti}
            \item \textcolor{gray}{Mahti}
            \item Roihu
            \item LUMI
        \end{itemize}
    \end{itemize}

\column{0.3\textwidth}
    \begin{itemize}
        \item Object storage
        \begin{itemize}
            \item Allas
            \item LUMI-O
        \end{itemize}
        \item Cloud platform
        \begin{itemize}
            \item cPouta
            \item ePouta
        \end{itemize}
    \end{itemize}

\column{0.3\textwidth}

\end{columns}

\end{frame}

\begin{frame}{Cluster vs Laptop}
\begin{columns}
\column{0.5\textwidth}
    Laptop
    \begin{itemize}
        \item Interactive
        \item Single version of software
        \item Few hard drives
        \item -
        \item GUI
    \end{itemize}

\column{0.5\textwidth}
    Cluster
    \begin{itemize}
        \item Batch processing with slurm
        \item Module system
        \item Parallel file system
        \item Data management
        \item Command line*
    \end{itemize}
\end{columns}
    
\end{frame}
