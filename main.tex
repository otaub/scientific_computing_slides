CSC's national HPC resources are free-of-charge for academic institutions, state research institutes, or the National Archives of Finland. The services discussed here are:
\column{0.5\textwidth}
    \begin{itemize}
        \item HPC clusters
        \begin{itemize}
            \item \textcolor{gray}{Puhti}
            \item \textcolor{gray}{Mahti}
            \item Roihu
            \item LUMI
        \end{itemize}
        \item Object storage
        \begin{itemize}
            \item Allas
            \item LUMI-O
        \end{itemize}
        \item Cloud platform
        \begin{itemize}
            \item cPouta
            \item ePouta
        \end{itemize}
    \end{itemize}
